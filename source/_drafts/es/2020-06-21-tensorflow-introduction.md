Bienvenido a este tutorial, el primero de una serie de tutoriales sobre TensorFlow, donde describiré cómo funciona TensorFlow, su arquitectura y cómo empezar a usarlo. Más adelante en esta serie, también trataré temas como el aprendizaje por transferencia, crear modelos personalizados y cómo implementarlos en diferentes plataformas (o dispositivos). Todo esto aplicado en varios dominios, como la visión por computadora, el procesamiento del lenguaje natural, el reconocimiento de voz, etc.

### Qué es TensorFlow?

TensorFlow es una biblioteca de código abierto desarrollada por el equipo de Google Brain, la cual nos proporciona una API intuitiva y fácil de usar (para python y c ++) para construir redes neuronales. Adicionalmente, tf nos ofrece un conjunto de herramientas, que facilitan no solo el desarrollo de nuestros modelos, sino también la implementación de ellos en diferentes dispositivos, tales como Tensorflow Hub, Tensorboard, Tensorflow Lite, etc. En el transcurso del curso, estaremos hablando de cada una de ellas.
