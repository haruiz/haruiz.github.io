### What is TensorFlow?

TensorFlow is an open-source  library developed by the Google Brain team {% cite 1 %}, which provides us an intuitive and easy-to-use API (for python and c ++) to build neural networks. In addition, it offers us a set of tools, which facilitates not only the development of our models, but also the implementation of them in different devices, such as Tensorflow Hub, Tensorboard, Tensorflow Lite, etc. During the course, we will be talking about each of them.

It was first made available under the Apache 2.0 License in November 2015 and has since grown rapidly; as of Jun 2020, its [GitHub repository](https://github.com/tensorflow/tensorflow) has more than 88,628 commits, with roughly 2,608 contributors.
